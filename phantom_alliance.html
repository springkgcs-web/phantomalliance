📜 Server Rules

1. No player killing (PvE server — KOS is strictly prohibited)
2. Respect all players and staff at all times — zero tolerance for toxicity
3. No griefing, trolling, or intentionally disrupting other players' gameplay
4. No hacking, cheating, or exploiting game bugs for personal gain
5. No blocking roads, military spawns, or loot areas with base structures
6. Base raiding is not permitted on this server
7. Follow all staff instructions — staff decisions are final
8. Any disputes must be raised via Discord — do not confront players in game
Breaking any rules may result in a kick, ban, or wipe depending on severity.
